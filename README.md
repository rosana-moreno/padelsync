# Padelsync V1

Tournament management app for padel tournaments. Built with React + TypeScript + Vite.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
